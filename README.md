# 🎥 TikTok Exact Replica (2026)

**Pixel-perfect static clone** of the official English TikTok homepage (https://www.tiktok.com/).

![Preview](https://picsum.photos/id/1015/800/400)

## ✨ Features
🎯 **Exact match** – colors (#010101 background + #ee1d52 pink), fonts, texts, layout  
📱 **Fully responsive** – desktop, mobile, tablet (Bootstrap 5 + mobile-first)  
🔥 **Smooth animations** – hover card lift, play icon pop, heart like animation  
❤️ **Interactive** – real-time like button with count increase  
♾️ **Infinite scroll** – auto-loads more videos  
🔍 **SEO optimized** – meta tags, semantic HTML, fast loading  
🛒 **TikTok Shop banner** + exact category chips (All, Singing & Dancing, etc.)  
📂 **Flat structure** – no folders, ready for GitHub Pages / Vercel  
🖼️ **Video grid** with 9:16 thumbnails and hover play  

## 🛠 Tech Stack
HTML5 • CSS3 • JavaScript • Bootstrap 5 • SVG
